第六章字典
6.1一个简单的字典
Python中字典是放在花括号{}里面的一系列 键—值 对。
cars={‘color’:’white’,’point’:5}
print(cars[‘color’])
print(cars[‘point’])
6.2使用字典
字典中每个键与之关联一个值，值可以为Python中的任何对象，可以通过键来访问值。
添加键值对：
cars={‘color’:’white’,’point’=5}
cars[‘length’]=7
创建空字典：
cars={}
修改字典中键对应的值：
cars[‘color’]=’yellow’
删除字典中的键值对，del语句指定字典和对应的键：
del cars[‘color’]
字典中存储一个对象的多个信息时，需要缩进“四个空格”：
Cars={
‘a’=1,
‘b’=2,
}
6.3遍历字典
遍历字典中的所有信息，即键值对：
for key,value in cars.items():
for语句中包含字典名称cars，同时包含方法items()，它返回一个键值对列表，该 列表中的键存储在变量key中，值存储在定义变量value中。
遍历字典中的所有键，用到方法keys()：
for key in cars.keys():
按照顺序遍历字典中的所有键，用到方法sorted()：
for key in sorted(cars.keys())
遍历之前，先将字典中的键按照顺序排列好。
遍历字典中的所有值，用到方法values()：
for value in cars.values():
如果要遍历字典中所有不重复的元素，可以使用集合set():
for value in set(cars.values())
集合类似于列表，但是集合中的元素都不重复。

6.4嵌套
列表中嵌套字典：

第七章用户输入和while循环
7.1函数input()的工作原理
函数input()让程序停止运行，等待用户输入，并将输入关联变量。
函数input()接受参数即向用户显示提示和说明：
a=input(“shuruwei: ”)
print(a)
用户输入的为字符类型，可以使用int()函数如：number=int(bianliang)来转换为数值类型。
求模运算符%将两个数相除，返回余数，如：a=5%4
7.2while()循环简介
while后接条件，然后加冒号，下面缩进之后是循环执行的部分。
while message!=’quit’:
在实际的程序中，常常添加标志，如：
active=True
while active:
break语句可以推出循环。
continue语句可以返回到循环开头。
7.3使用while()循环来处理列表和字典
删除包含特定值的所有列表中的元素，如cars为一个列表，其中元素car多次出现：
while ‘car’ in cars:
cars.remove(‘car’)
使用用户输入可以while循环填充字典。
第八章函数
8.1定义函数
关键字def加函数名字，加圆括号和括号内函数需要的参数信息，加冒号结尾，之后缩进行构成了函数体，如：
def hanshu():
print(“我是函数”)
像函数传递信息，则需要在定义的圆括号中加入参数信息，如：
def hanshu(canshu):
print(“函数”)
8.2传递实参
位置实参：关联方式是基于形参的位置和顺序，如：
def number(n1,n2):
print(n1+n2)
number(‘a’,’b’)
关键字实参：其中每个实参都是由变量名和值组成，如：
def number(n1,n2):
print(n1+n2)
number(n1=’a’,n2=’b’)
函数定义中的形参可以加入默认值，但要把有默认值的形参放在无默认值的形参之后。
8.3返回值
函数可以永return语句返回，将值返回到调用函数的代码行。
函数可以让实参变为可选的，可以给实参指定一个默认值—空字符串，并将其移到形参列表的末尾：
def hanshu(a,b,c=’’):
函数也可以返回任意类型的值，包括列表与字典。
8.4传递列表
将列表作为实参传递给函数时，函数便可以直接访问列表中的内容，在很多程序应用中都可以提高效率，如：
def hanshu(names):
for name in names:
函数传递列表作为实参时可以在函数体中对列表进行修改，但是也可以使列表在函数使用中不会被修改，即可以向函数中传递列表的副本，如：
function(list[:],bianliang)
8.5传递任意数量的实参
函数可以使用一个形参toppings，则无论调用语句传了多少个实参，函数都可以接纳进来，其实形参toppings中的星号*是让Python创建一个名为toppings的空元组，将所有收到的实参都收纳到元组之中，即：
def hanshu(toppings):
Print(toppings)
Hanshu(‘one’,’two’,’three’)
函数使用可以结合位置实参和任意数量实参，如：
def hanshu(weizhi,toppings):
函数中使用任意数量的关键字实参，其中双星号是让Python创建一个指定名字的空字典，并将函数调用所有传进来的键值对实参都封装到创建的这个空字典中，在这个字典中可以像访问其他字典一样访问定义的字典实参，如：
def hanshu(weizhi,**jianzhidui):
8.6将函数存储在模块中
Python中的导入必须首先创建.py类型的模块文件，然后可以：
导入整个模块： import mokuaiming
导入模块中的函数： from mokuaiming import hanshu1,hanshu2,hanshu3
使用as给函数指定一个别名： from mokuaiming import hanshu as hanshubieming
使用as给模块指定一个别名： import mokuaiming as mokuaibieming
导入一个模块中所有函数（不建议使用）： from mokuaiming import *
第九章类
9.1创建和使用类
class Dog():
“””一个小狗的类”””
def init(self,name,age):
“””初始化属性name和age”””
self.name=name
self.age=age
def sit(self):
print(“dog sit”)
def roll_over(self):
print(“dog roll over”)
其中self.name=name，以self为前缀的变量都可以供类中所有的方法使用，也可以通过类的任何实例来访问这些变量，该语句将获取存储在形参name中的值，并将值关联到变量name中，该变量name也通过self关联到了当前创建的实例，像这样可以通过类的实例访问的变量叫做属性。
Python中根据类来创建实例：
class Dog():
def init(self,name,age):
…
my_dog=Dog(‘pipi’,2)
该行代码中，Python使用实参’pipi’和2来调用类的_init_方法，该方法创建一个Dog类的实例，并且通过使用我们提供的值来设置属性name和age。
访问类实例的属性，用句点.访问，如：my_dog.name
类实例化后也可以通过实例用句点.调用类中的方法，如：my_dog.sit()
9.2使用类和实例
9.3继承
子类的_init_方法：
Class zilei(fulei):
Def init(self,a,b):
Super().init(a,b)
…
Super()是个特殊的方法，将超类与子类关联起来。
在Python中，让子类继承父类之后，可添加区分子类和父类的新属性和方法，子类也可以重写父类的方法。
在实际应用中，有时发现给一个类添加的细节越来越多，其中的属性和方法清单以及文件都会越来越长，故我们可以将这个类的某一个部分单独拿出来作为一个类，并可以将这个新组成的类的实例作为其他类的一个属性。
9.4导入类
Python中导入类和导入函数的语法基本一致。
9.5Python标准库
Python标准库即表示一堆Python中已存在的模块，我们可以导入这些模块并调用其中的类或函数，如模块collections中的OrderedDict类：
From collections import OrderedDict
shili=OrderedDict()
…
上述代码，一个是导入OrderedDict类，该类即表示一个记录了键值对添加顺序的字典，第二行表示的该类的实例化。
第十章文件和异常
10.1从文件中读取数据
读取整个文件：
with open(‘xxx.txt’) as filename:
contents=filename.read()
print(contents)
函数open(‘xxx.txt’)打开一个文件名为xxx.txt的文件，并返回一个表示文件的对象，再由as将该文件对象存储到后面变量filename中，接着下一行代码表示对该存储文件的变量执行方法read()读取文件的全部内容，并将读取到的全部内容作为一个长字符串存储到变量contents中，接着打印该变量的值。（注意read()方法到达文件末尾时返回一个空字符串。）
文件路径由绝对路径也有相对路径。
Python中创建一个包含文件各行内容的列表：
with open(filename) as file_object:
lines=file_object.readlines()
for line in lines:
print(line)
Readlines()方法从文件中读取每一行，并将其存储在一个列表中，即该方法是返回一个列表的，故我们上述代码，lines=file_object.readlines()可以将文件中的每一行都读取，作为一个列表，存储在变量lines中，该变量就表示了一个存储文件每一行内容的列表。
10.2写入文件
Python中写入空文件：
with open(filename,’w’) as file_object:
file_object.write(“把这些写入该空文件”)
若给文件中添加内容，则使用附加模式打开文件，这样写入文件的每一行都自动添加到文件的末尾，如：
with open(filename,’a’) as file_object:
file_object.write(“把这些添加到文件末尾”)
10.3异常
处理异常问题使用try-except代码块，try-except-else代码块。
Python中方法split()，是根据一个字符串创建一个单词列表，该方法以空格为分隔符将字符串分为单词组成，创建并返回一个单词列表，如：title.split() 其中title为一个字符串。
Python中使程序处理异常一声不吭继续运行的为pass语句，需要单独放在except代码块中。
10.4存储数据
模块json可以将简单的Python数据结构转存到文件中，并在程序再次运行时加载文件中的数据，还可以使用json在Python程序之间分享数据。
json.dump()的使用：
json.dump()需要两个实参，一个要存储的数据，一个为用于存储数据的文件对象。
import json
Numbers=[5,15,25]
Filename=”numbers.json”
With open(filename,’w’) as file_object:
Json.dump(numbers,file_object)
json.load()的使用：
Json.load()将文件对象存储的内容读取到内存中。
import json
Filename=”numbers.json”
With open(filename) as file_object:
Numbers=Json.load(file_object)
Print(numbers)
